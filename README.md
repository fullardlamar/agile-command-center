# 🏃 Agile Command Center

An interactive Scrum Master and Product Owner dashboard demonstrating Agile project management skills for DoD and federal agency environments.

## 🌐 Live Demo
👉 **[Click here to see the Agile Command Center](https://fullardlamar.github.io/agile-command-center)**

## 📊 Features
| Feature | Description |
|---------|-------------|
| Sprint Board | Drag and drop user stories between To Do, In Progress, and Done |
| Product Backlog | Create and manage user stories with priority and story points |
| Burndown Chart | Visual chart showing ideal vs actual sprint progress |
| Metrics Dashboard | Team velocity, sprint health, and stories completed |
| Sprint Report | Auto-generated stakeholder report with recommendations |

## 🎯 Agile Concepts Demonstrated
- Sprint planning and backlog management
- User story creation with story points
- Burndown chart tracking
- Team velocity measurement
- Stakeholder reporting
- SAFe Agile principles

## 🔧 Tech Stack
| Tool | Purpose |
|------|---------|
| HTML/CSS | Structure and styling |
| JavaScript | Interactive functionality |
| Chart.js | Burndown and metrics charts |
| GitHub Pages | Live deployment |

## 🚀 How to Run
Simply visit the live demo or clone the repo:
```bash
git clone https://github.com/fullardlamar/agile-command-center.git
cd agile-command-center
open index.html
```
